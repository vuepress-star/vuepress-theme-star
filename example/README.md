### As Easy as 1, 2, 3

:::: code-group

::: code-group-item title="NPM" active

```bash
# install in your project
npm install -D @starzkg/vuepress-theme-star

# create a markdown file
echo '# Hello VuePress Theme Star' > README.md

# start writing
npx vuepress dev

# build to static files
npx vuepress build
```

:::
